Design Tokens

The smallest pieces of a design system — basically anything that can be described using letters and numbers.

No more manually picking values from design files!
Use these SASS variables and mixins (= design
tokens) when you write your styles. This is
basically "design translated into code":

.myComponent {
  color: $lightmode-color-text;
  @include defaultBorderRadius();
}

Utility Classes

Functional CSS classes that serve one particular purpose.

Compose layouts without writing CSS:

m:flx   = display:flex from breakpoint M and up
m:w-1-2 = width:50% (1/2) width from breakpoint M and up
p-4     = padding of factor 4 on all sides

<div class="m:flx">
    <div class="m:w-1-2 p-4">1</div>
    <div class="m:w-1-2 p-4">2</div>
</div>
Use the utility classes in your project via CDN:
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/adjust-design-system/design-tokens/dist/utilities.min.css" />
