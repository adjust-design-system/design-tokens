Color

These tokens represent all colors in the UI – text, background, icons, etc. There are three main colors with 16 shades each.

Alias Color Tokens

These tokens have names which communicate their intended and purpose. They mostly reference other color tokens, therefore the term alias.

#08d3ff $color-brand The Adjust brand color
#ffffff $color-white Plain white
#0b0d18 $color-black The darkest UI color
#1a62ff $color-focus For focussed input elements
#353a52 $lightmode-color-text Default text color in light mode
#f4f5f9 $darkMode-color-text Default text color in dark mode

Please Notice: As the team is currently finishing up the UI design language, more alias tokens will be available soon.
